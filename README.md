# 小耳朵 astrbot_plugin_audio_ears

AstrBot 语音 & 音频处理插件。让 Bot 能听懂语音消息、分析音乐、学习主人声音特征。

## 功能

### 语音识别

收到语音消息时自动处理，支持四种模式：

| 模式 | 说明 |
|------|------|
| 省钱转写 | 只输出文字转写 |
| 小耳朵报告 | 转写 + 声音形状描述 |
| 直听模式 | 原音频直接挂给主模型听 |
| 自动模式（默认） | 主模型支持音频就直听，否则走报告 |

### 声音底纹

学习主人正常说话的声音特征，之后每次收到语音自动对比变化。

**使用方法：**

1. 发几条正常说话的语音（建议 5 条以上）
2. 每发完一条，发送指令：`收声音样本`
3. 之后收到语音时，自动附上声纹变化信息（音高、响度、语速等）

**指令：**

| 指令 | 作用 |
|------|------|
| `收声音样本` | 把最近一条语音收为底纹样本 |
| `声音底纹` | 查看当前底纹状态 |
| `声音对比` | 手动对比最近一条语音和底纹 |

### 音乐分析

通过音乐卡片分享或音频文件发送歌曲时，自动进行听觉分析（曲风、情绪、结构等）。

- 音乐卡片：QQ音乐/网易云分享自动触发
- 音频文件：本地判定为音乐时后台异步分析，不阻塞回复

**指令：**

| 指令 | 作用 |
|------|------|
| `听歌报告` | 查看最近一次音乐听觉报告 |

## 安装

将插件文件夹放入 AstrBot 插件目录即可。

**依赖：** 无额外依赖，使用 AstrBot 自带环境。

**可选：** 系统安装 ffmpeg 可获得更好的音频格式兼容性。未安装时插件会尝试自动处理常见格式。

## 配置

安装后在 AstrBot 管理面板的插件配置页调整。所有配置项均有中文说明和提示。

核心配置：

- **听法档位** — 选择语音处理模式
- **语音听觉LLM提供商** — 用于报告模式的模型
- **群聊也听语音** — 默认仅私聊，可开启群聊
- **音乐听觉** — 音乐分析相关设置

## 从旧版升级

直接用新插件文件夹替换旧版即可。旧的声音底纹样本文件仍然有效，插件会自动读取。
